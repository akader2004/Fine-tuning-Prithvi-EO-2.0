# Fine-tuning Prithvi-EO-2.0

This repository contains the evaluation notebook for comparing a Random Forest baseline against an available Prithvi-EO-2.0 checkpoint on the HLS burn scar segmentation dataset.

Due to limited RAM and compute resources, the Prithvi model could not be fully fine-tuned. The reported Prithvi results should be interpreted as checkpoint evaluation results, not the full capability of Prithvi-EO-2.0.

**The Prithvi fine-tuning notebook was adapted from a publicly/shared notebook by Blumenstiel. Our group used it as the starting point for evaluating the available Prithvi-EO-2.0 checkpoint and comparing it against a Random Forest baseline.**
